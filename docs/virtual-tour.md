# Triton Virtual Tour Script

Use this script for demos, onboarding sessions, and stakeholder walkthroughs.

## Tour Objective

Show how Triton turns coastal intelligence into operational action in under 15 minutes.

## Setup Checklist

- API and web app are running
- At least one beach is selectable
- Browser is zoomed for presentation
- Fullscreen button is visible in header

## Demo Script

1. Start in Dashboard.
- Explain Beaches Online and Severe Alerts counters.
- Turn fullscreen on for command-center view.

2. County filtering and map triage.
- Switch between counties.
- Select a severe marker and explain severity legend.

3. Beach Intelligence deep dive.
- Show TSI arc and trend charts.
- Call out live wind/wave/tide context.

4. Forecast walkthrough.
- Open Forecast tab.
- Compare 24h and 7d horizons.
- Highlight top driver explainability.

5. Field observer workflow.
- Open Observer tab.
- Submit a sample observation.
- Mention optional photo upload path.

6. Camera verification.
- Open Cameras tab.
- Search a county and switch active feed.

7. Reporting handoff.
- Open Reports tab.
- Show regional summary and beach detail report.

8. Finish with controls.
- Toggle Light/Dark mode.
- Exit fullscreen.

## Talking Points by Persona

- Operators: faster triage and deployment prioritization
- Leadership: county-level cost/risk picture
- Procurement: contract and compliance context
- Field teams: direct observation to command feedback loop

## Demo Close

- Summarize top 3 value points for audience
- Confirm next rollout milestone and success metrics
